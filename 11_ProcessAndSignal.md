# 3. 启动新进程

## system函数

    # include <stdlib.h>
    
    int system(const char* string);
    
函数作用：运行传递给它的命令，命令执行情况类似于：

    $ sh -c "command"
    
### 返回值

- 无法启动shell，返回127
- 其他错误，返回-1
- 执行成功，返回命令退出码
